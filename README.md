# stock-market-analysis
# Stock Market Risk and Portfolio Analytics

This project analyses historical stock market data to investigate the relationship between risk and return in financial markets. Using Python, SQL, and statistical techniques, the project collects and processes market data, calculates key risk metrics, explores portfolio diversification strategies, and evaluates machine learning models for predicting stock price movements.

## Key Features

* Collection and storage of historical stock market data
* Exploratory data analysis and visualisation of market trends
* Calculation of financial metrics including returns, volatility, Sharpe ratios, and drawdowns
* Correlation analysis between stocks and market indices
* Portfolio optimisation using modern portfolio theory
* Machine learning models for stock direction prediction
* Interactive dashboard for analysing market performance and portfolio risk

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib / Plotly
* SQL
* Scikit-learn
* Streamlit

## Objectives

The primary goal of this project is to apply statistical, mathematical, and computational techniques to real-world financial data while demonstrating practical data analysis, visualisation, and modelling skills.
